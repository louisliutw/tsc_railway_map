# 1978 年台糖鐵路

依據<<1978年臺糖鐵道平面、分佈圖>>所載，描繪 1978 年台灣糖業鐵道地圖。
所有路線、場站位置都依據參考資料所載的位置和里程描繪。
有疑慮的內容會在會依各場路線在下方說明。

## 各廠說明

### 高雄
* 不確定太爺線、石案潭線在糖廠附近路線位置。這裡一帶鐵路有改線過，平面圖的里程數據應該是改線前的數據。但是<<高雄市舊航照影像(1970年代)>>所記錄到的影像是改線後的。
* 太爺線各路段里程多半與平面圖所記載不同。
* 左營線九甲圍在九甲圍聚落旁，但在鐵道平面圖中記載為 "六甲圍"，應該是鐵道平面圖的筆誤。
* 根據 [藍田踏查 - 追追追](http://blog.cshs.kh.edu.tw/index.php?op=ViewArticle&articleId=1265&blogId=313)，左營線下鹽田路段的位置在藍田路旁，而不是沿著新庄營區圍牆。
* 梓官線在洞案厝之後的路線少有地圖記載，<<1987-台灣地形圖-1:100,000>> 有描繪，但是在大舍甲的走法與平面圖不同。依據假名留先生的記憶，梓官線在梓官路段現為台17線。

### 小港
* 淡水線裝車場位置都是依照里程所標記，但是在 高雄市舊航照影像(1970年代) 中沒有看到明顯設施。
* 在<<高雄市舊航照影像(1970年代)>> 有看到一條分歧往現在大發工業區的路線，但當時大發工業區已經開發，所以採用 <<台湾 臺糖鐵道 廃線跡 配線図式路線図>> 的見解描繪大寮側線與大寮農場。
* 三角線長度較平面圖短。

### 旗山
* 大部分路線是依據高雄百年歷史地圖的 <<高雄市舊航照影像(1970年代)>>描繪。
* 依據王翔的調查，磚子磘線跨越省鐵橋墩現為供水管用。
* 旗尾線依據平面圖所記載的里程，路線還保留往美濃方向的一小段路線。

### 月眉
* 大甲線 4、5 號裝車場之間距離較大，但是 5、6、7 裝車場距離很小，是不是月眉糖廠鐵道平面圖的資料有誤？
* 中和線末端位置是依據里程標定，但不確定是否的確通過現今高鐵下方。

### 仁德
* 1977 年路竹交流道通車，所以阿蓮線與一甲裝車場的位置依據 [此文](http://blog.xuite.net/tropicofcancer/2327/20742124-%E4%BB%81%E5%BE%B7%E7%B3%96%E5%BB%A0%E9%98%BF%E8%93%AE%E7%B7%9A3%E4%B9%8B3)與地籍圖描繪。
* 1977 年高速公路已通車，二苓線迂迴穿過高速公路，但是平面圖里程未更新。
* 關廟線終點附近路線依據平面圖記載的路線長度以及<<台糖地籍資料>>、台南百年歷史地圖 WMTS 服務的<<台南舊航照影像(1975,高經度)>>描繪。
* 沙汰線刣豬厝、東楊桃、西楊桃的位置有待進一步的考證。
* 二甲線描繪長度較平面圖長。
* 本洲線水車、路竹農間里程略長於平面圖；本洲線全線描繪長度比平面圖長。
* 不確定中路社詳細位置。

### 永康
* 大部分路線依據台南百年歷史地圖的<<台南舊航照影像(1975,高經度)>>描繪。
* 在本圖最早的版本中，將<<台南舊航照影像(1975,高經度)>> 中的一個彎道路線描繪為太爺線。但根據徐貳帆先生所收藏的<<永康糖廠運輸課路線圖>>以及 Dafydd Fell 先生拍攝的照片判斷，該路線應該為新化線。不知道是否太爺線存在？在進一步考證之前，還是先先將太爺線的資料移除。
* 道爺線新市線起點、看西線起點間較平面圖所記載的長度還長。

### 善化

* 倉庫線位置依照 <<正射影像圖>> 描繪。
* 不確定六分寮線起點位置，有待進一步考證。
* 南科附近路線因土地重劃確切位置不確定，路線皆依<<經建一版>>描繪。
* 新化幹線道爺、頂潭支線訖點描繪的里程較平面圖記載略長。
* 安定幹線新化幹線起點後到終點的長度比平面圖記載長。所以後半段的裝車場位置也有偏差。

### 北港
* 蚵寮線在蚵寮裝車場後的路線都是推估，不確定位置。百年歷史地圖中 <<1987-台灣地形圖 1:100,000>> 有路線繞過蚵寮聚落南邊，在地籍圖剛好有一帶狀但狹窄的土地。另外，地籍圖在聚落北邊有疑似鐵路弧線。如果路線是經由聚落南邊，線型較接近地形圖，且路線終點會在金湖聚落外。如果路線是經由蚵寮北邊，路線外觀較接近平面圖所描繪，但路線終點要在金湖聚落內，路線長度才會符合平面圖所記載。因為地形圖的記載，以及路線終點和裝車場的位置，所以本圖採取南邊路線描繪。
* 四湖線大部分路線因土地重劃，沒有辦法找出明確位置，只能依舊地圖搭配地藉圖推測位置。另外因平面圖有 "四湖線迄點（銜接下寮線起點）"，所以假設 1977 年時路線已經截短。
* 溪底線後段因土地重劃，沒有辦法找出明確位置。依路線長度推估終點位置。
* 子茂線後段因土地重劃，沒有辦法找出明確位置，路線為描繪<<經建一版>>。在<<經建一版>>中，子茂線已經成為一條小徑。
* 內寮線、後溝子線因土地重劃，沒有辦法找出明確位置，路線依<<1956-台灣地形圖>>參考地籍圖描繪。
* 內寮支線因土地重劃，沒有辦法找出明確位置。起點位置是依內寮線里程推測，但路線走向沒有任何參考資料，純粹是依地籍圖紋理猜測。
* 溪底後段因土地重劃，沒有辦法找出明確位置。位置參考<<1956-台灣地形圖>>並依據里程描繪。
